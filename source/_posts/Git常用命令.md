常用的一些 Git 命令如下：

1. **git init**: 初始化一个新的 Git 仓库。
2. **git clone [仓库地址]**: 克隆一个远程 Git 仓库到本地。
3. **git add [文件名]**: 将文件添加到暂存区，准备提交。
4. **git commit -m "[提交消息]"**: 提交暂存区的文件到本地仓库，附带一条提交消息。
5. **git status**: 查看工作区、暂存区和本地仓库的状态。
6. **git log**: 查看提交历史记录。
7. **git branch**: 查看本地分支列表。
8. **git checkout [分支名]**: 切换到指定分支。
9. **git checkout -b [新分支名]**: 创建并切换到新的分支。
10. **git merge [指定分支名]**: 合并指定分支到当前分支。
11. **git pull**: 从远程仓库拉取最新的代码。
12. **git push**: 推送本地代码到远程仓库。
13. **git remote -v**: 查看远程仓库列表和地址。
14. **git reset [文件名]**: 从暂存区移除指定文件，但保留在工作区。
15. **git reset --hard [提交哈希]**: 将仓库回滚到指定的提交。
16. **git stash**: 将当前未提交的修改暂存起来。
17. **git cherry-pick [提交哈希]**: 将指定提交的修改应用到当前分支。
18. **git fetch**: 从远程仓库获取最新的分支列表和提交，但不自动合并。
19. **git remote add [远程仓库名] [仓库地址]**: 添加一个新的远程仓库。
20. **git remote remove [远程仓库名]**: 移除指定的远程仓库。

可以使用 `git --help` 或 `git [命令] --help` 来查看 Git 命令的详细帮助文档。
